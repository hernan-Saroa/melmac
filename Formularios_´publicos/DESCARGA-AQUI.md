# 📦 Instrucciones de Descarga - MelmacCreditForm

## Archivos Disponibles

Tienes dos opciones para descargar el proyecto completo:

### Opción 1: Archivo ZIP (Recomendado para Windows)
**Archivo:** MelmacCreditForm.zip (41 KB)
**Descomprimir con:** WinZip, 7-Zip, Windows Explorer

### Opción 2: Archivo TAR.GZ (Recomendado para Linux/Mac)
**Archivo:** MelmacCreditForm.tar.gz (34 KB)
**Descomprimir con:** tar -xzf MelmacCreditForm.tar.gz

## Pasos Después de Descargar

### 1. Descomprimir el archivo
```bash
# Para ZIP (Windows/Mac/Linux)
unzip MelmacCreditForm.zip

# Para TAR.GZ (Linux/Mac)
tar -xzf MelmacCreditForm.tar.gz
```

### 2. Entrar al directorio
```bash
cd MelmacCreditForm
```

### 3. Instalar dependencias
```bash
npm install
```

### 4. Iniciar en desarrollo
```bash
npm run dev
```

Abre tu navegador en: http://localhost:3000
