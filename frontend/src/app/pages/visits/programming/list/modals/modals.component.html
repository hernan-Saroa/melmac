Generar programación
PROGRAMACIÓN MANUAL Realizara la asignación de las tareas 1 a 1 segun la lista que se encuentre en tareas por asignar o en la lista de tareas por reasignar.
PROGRAMACIÓN AUTOMATICA El sistema ayudara asignando tareas automaticamente donde el usuario solo debe validarlas en la lista de tareas asignadas y asignar manualmente las que no se pudieron programar.
